# 前言

欢迎来到基于微信平台的购物商城小程序开发项目。本项目旨在为广大开发者提供一个微信小程序商城的快速开发解决方案。以下是本项目的详细说明。

## 内容介绍

本项目是一个基于微信平台的购物商城小程序，主要包括商品展示、商品分类、购物车、订单管理、个人中心等功能模块。通过使用微信小程序，用户可以更便捷地浏览和购买商品，同时，商家也可以更好地管理和运营商城。

## 技术介绍

本项目采用以下技术栈：

- **语言：Java**
- **使用框架：Spring、Springmvc、MyBatis、微信小程序**
- **前端技术：JS、Vue、CSS3、Uniapp**
- **开发工具：IDEA/Eclipse，Uniapp**
- **数据库：MySQL 5.7/8.0**
- **数据库管理工具：phpstudy/Navicat**
- **JDK版本：jdk1.8**
- **Maven: apache-maven 3.8.1-bin**
- **前端环境：Node.Js 12\14\16**
